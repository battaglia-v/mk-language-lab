# Play Store Graphics

This folder contains the core Play Store listing assets.

- icon-512.png (512x512 PNG, app icon)
- feature-graphic-1024x500.png (1024x500 PNG)
- screenshots/ (phone screenshots from production)
- screenshots-legacy/ (previous captures archived)

Notes:
- The screenshots are captured from the live site via `scripts/capture-store-screenshots-v2.mjs`.
- Current screenshot dimensions: 1080x1920 (light theme).
- The Play Store icon is synced from `android-twa/store_icon.png`.
