## How to Add This to GitHub

1. Go to your repository: https://github.com/battaglia-v/mk-language-lab
2. Click the **⚙️ Settings** gear icon (top right, next to About section)
3. In the "Description" field, paste the description above
4. In the "Website" field, add your deployment URL
5. In the "Topics" field, add the suggested tags (select from autocomplete or type custom)
6. Check "Releases" if you plan to use GitHub Releases
7. Check "Packages" if you publish npm packages
8. Click "Save changes"

## Social Preview Image (Optional but Recommended)

GitHub allows you to set a social preview image that appears when sharing your repo:

1. Go to Settings → Options → Social preview
2. Upload an image (1280×640px recommended)
3. Good candidates:
   - Screenshot of your app's homepage
   - Logo with app name
   - Feature showcase graphic

This image appears in:
- Social media shares (Twitter, LinkedIn, etc.)
- GitHub search results
- GitHub Explore page
