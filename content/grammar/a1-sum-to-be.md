# A1: Verb "To Be" (сум)

## Conjugation
- јас сум
- ти си
- тој/таа/тоа е
- ние сме
- вие сте
- тие се

## Usage
Use "сум" for identity, origin, and states.
Third-person singular uses the same form for all genders: тој е, таа е, тоа е.

Examples:
- Јас сум студент.
- Ти си уморен.
- Тој е доктор.
- Таа е од Скопје.
- Тоа е ново.
- Ние сме уморни.
- Вие сте од Охрид.
- Тие се подготвени.

## Notes
- Subject pronouns can be dropped if context is clear.
- Negation: не сум, не си, не е, не сме, не сте, не се.
