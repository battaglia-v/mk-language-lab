# A1: Alphabet & Pronunciation

## Goal
Build confidence with the Macedonian Cyrillic alphabet and special letter sounds.

## Key Letters and Sounds
- Ѓ / ѓ (gje) — ѓердан, меѓу
- Ќ / ќ (kje) — ќерка, куќа
- Љ / љ (lje) — љубов, љубезен
- Њ / њ (nje) — њушка, њива
- Џ / џ (dzhe) — џем, џеб
- Ѕ / ѕ (dze) — ѕвезда, ѕид

## Practical Tips
- Read letters out loud in pairs: ќ/ч, ѓ/г, љ/л, њ/н.
- Watch for minimal pairs: куќа vs куче, ѓавол vs гавол.
- Stress is mobile; focus on clear consonants first.

## Mini Practice
- Write three words that include Љ/Њ/Џ.
- Read these aloud: ѕвезда, ќерка, љубов.
