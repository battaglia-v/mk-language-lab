# A2: Possessives (мој, твој, ...)

## Agreement
Possessives agree with the noun's gender and number:
- мој / моја / мое
- твој / твоја / твое
- негов / негова / негово
- нејзин / нејзина / нејзино
- наш / наша / наше
- ваш / ваша / ваше
- нивен / нивна / нивно

## Examples
- мој брат
- твоја сестра
- негово дете
- нејзина книга
- наша куќа
- ваш автомобил
- нивниот план
- нивните пријатели

## Notes
Plural forms add -и/-те (моите, твоите, неговите, нејзините, нашите, вашите, нивните).
Definite forms add a suffix: мојата куќа, твоето дете, неговиот автомобил.
