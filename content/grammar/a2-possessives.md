# A2: Possessives (мој, твој, ...)

## Agreement
Possessives agree with the noun's gender and number:
- мој / моја / мое
- твој / твоја / твое
- негов / негова / негово
- нејзин / нејзина / нејзино
- наш / наша / наше
- ваш / ваша / ваше
- нивен / нивна / нивно

## Examples
- мој брат
- моја сестра
- мое дете
- нивните пријатели
