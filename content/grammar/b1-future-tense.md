# B1: Future Tense (ќе)

## Form
- ќе + present tense: ќе одам, ќе читам
- Negative: нема да + present tense
- Example (оди): јас ќе одам, ти ќе одиш, тој/таа/тоа ќе оди, ние ќе одиме, вие ќе одите, тие ќе одат

## Examples
- Јас ќе одам дома.
- Ти ќе одиш утре.
- Тој ќе оди на работа.
- Таа ќе оди во Скопје.
- Тоа ќе биде важно.
- Ние ќе одиме во Охрид.
- Вие ќе одите рано.
- Тие нема да одат денес.
- Дали ќе дојдеш?

## Notes
Use time markers (утре, следната недела) to clarify future meaning.
The particle "ќе" stays the same for all persons.
